Abstract: 
Anthropometry is known as the science that studies the human body dimensions, acquired using special devices and techniques whose results are analyzed through statistics. Recently, image processing and hardware advances are allowing the development of applications that allow an user to preview different types of environments according to the user measurements. This project proposes the development of a complimentary tool for acquiring user anthropometric data for characterizing users in the Mil. Nueva Granada University in Colombia, South America using Microsoft???s Kinect skeletal tracking for developing and assess the design of workspaces in several areas such as laboratories.
