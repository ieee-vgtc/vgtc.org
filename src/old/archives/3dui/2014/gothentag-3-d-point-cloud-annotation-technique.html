Abstract: 
Go'Then'Tag (GTT) is a tool-set that helps users to edit complex 3-D data sets, and tag them at different levels. It allows to quickly select a subset of points in the 3-D point cloud and edit a tag associated to it. Our framework also comes with a tool to edit and manipulate a tree of tags, that allows overlapping hierarchies of tags. The interaction is handled through a tracked multi-touch device that combines 2-D and 3-D interaction techniques in a single device.
