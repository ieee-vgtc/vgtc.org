Abstract: 
This year's 3DUI challenge is a classic problem that, if solved correctly, can be conveniently used in a wide range of 3D virtual reality applications and environments. Our approach considers a robust solution based on bi-manual interaction gestures. We provide a first-person navigation mode for data exploration, point selection and annotation coupled with a bi-manual gesture interface, where the user is able to perform simple but powerful gestures to navigate through the 3D point cloud data within a 3D modeling tool to explore, select and/or annotate points.
